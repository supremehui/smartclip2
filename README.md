# 🚀 SmartClip (智能剪贴板助手)

**SmartClip** 是一款专为 Android 打造的效率增强工具。它能够智能嗅探剪贴板内容，根据预设规则实现一键直达目标 App 的深度链接（Deep Link），彻底告别手动复制、切换 App、等待弹窗的繁琐流程。

---

## ✨ 核心特性

* **⚡ 灵动岛式交互**：采用 1.7 倍放大的胶囊型浮窗设计，具备毛玻璃质感与 Q 弹的入场动画。
* **🔗 深度直达引擎**：支持淘宝、京东、抖音、闲鱼等主流 App 的 URL 提取与协议级跳转。
* **🧩 动态规则系统**：内置可视化规则编辑器，支持通配符匹配，可无限扩展自定义 App。
* **🌐 云端口令解析**：支持通过 API 密钥解析复杂的纯符号淘口令，实现无感直达商品页。
* **🛡️ 智能环境感知**：自动识别当前前台应用，若用户已在目标 App 内则智能静默，不触发干扰弹窗。

---

## ⚙️ 规则编辑指南

SmartClip 的强大核心在于其高度自由的规则系统。进入 **“规则编辑器”** 即可进行个性化配置：

### 1. 通配符匹配
所有的规则都支持使用 `*` 作为通配符。
* **示例**：`*省*市*` 或 `*路*号*`
* **逻辑**：只要剪贴板内容包含匹配的关键词，即可触发相应的地图导航或识别动作。

### 2. 自定义 App 跳转
你可以通过简单的语法让 SmartClip 支持手机里的任何 App。
* **语法格式**：`识别关键字*:App包名`
* **操作方式**：
    1. 点击“添加新应用规则”生成独立方框。
    2. 输入如：`查快递*顺丰:com.sfg.view`。
    3. 复制包含“查快递”的文本，点击气泡即可瞬间拉起顺丰速运。

### 3. 电商高级配置
针对淘宝纯口令（如 `￥...￥`），SmartClip 支持自定义 API 解析：
* **接口地址**：可自由填写第三方解析服务地址。
* **API Key**：私密存储，保护你的个人密钥不被泄露。
